## Explain the difference between mutable and immutable objects.
- **Mutable Objects**: Can be altered after creation.  
  Example:
  ```
  my_list = [1, 2, 3]
  my_list[0] = 0  # Modifying an element in the list
  print(my_list)  # Output: [0, 2, 3]
 ```
